URL Shortening 
================

### Overview

**URL Shortener** - **PHP : |Slim Framework + Twig + Paris & Idiorm|** Application to shorten your long URLs.
This application also includes multilingual support (i18n).


## Server Requirements

PHP 5.4.0 or newer.
